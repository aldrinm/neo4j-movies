3 Configure GORM for Neo4j

Firstly, this guide is built with GORM 6.1.0.BUILD-SNAPSHOT, so in order to complete the guide we must first configure Grails to use GORM 6.1.0.BUILD-SNAPSHOT.

To do so set the gormVersion property in gradle.properties to 6.1.0.BUILD-SNAPSHOT:

gradle.properties
grailsVersion=3.2.7
gormVersion=6.1.0.BUILD-SNAPSHOT
neo4jVersion=3.1.2
gradleWrapperVersion=3.0

build.gradle already contains a dependency to neo4j plugin. You will need to modify the neo4j dependency to also use GORM 6.1.0.BUILD-SNAPSHOT:

gradle.properties
    compile "org.grails.plugins:neo4j:${gormVersion - '.RELEASE'}"

In addition, GORM 6.1 no longer includes the grails-datastore-simple dependency so you need to exclude it from being resolved transitively in build.gradle:

gradle.properties
configurations.all {
    exclude module:'grails-datastore-simple'
}
dependencies {
...
}

Now modify the already existing neo4j configuration block within the grails-app/conf/application.yml file. Configure the Neo4j connection settings to use the "movies" password you set earlier:

grails-app/conf/application.yml
grails:
    neo4j:
        url: bolt://localhost
        options:
            cache_type: soft
        username: "neo4j"
        password: "movies"
